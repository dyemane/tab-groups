## Data Collection

Project Tab Groups does **not** collect, transmit, or share any user data. All data stays on your device.

## What Data Is Stored

- **Project configurations** (tab group names, colors, tab URLs and titles) are stored locally in `chrome.storage.local` on your device.
- **No data leaves your browser.** There are no servers, analytics, telemetry, or network requests of any kind.

## Permissions Used

| Permission | Why |
|-----------|-----|
| `tabGroups` | Read and manage Chrome tab groups (save, restore, switch) |
| `tabs` | Query, create, and organize tabs within groups |
| `storage` | Persist saved projects locally across browser sessions |

## Third-Party Services

None. This extension makes zero network requests.

## Export/Import

When you export projects, a JSON file is saved to your local Downloads folder. When you import, the file is read locally. No data is uploaded anywhere.

## Open Source

The full source code is available at [github.com/dyemane/tab-groups](https://github.com/dyemane/tab-groups). You can verify these claims by reading the code.
